# **App Name**: UniApply Hub

## Core Features:

- Student Request Submission: Employees can submit university application requests on behalf of students, including all necessary details and documents.
- Application Tracking: Track the status of each student's application with clear milestones and updates.
- Admin Dashboard: Admin users can oversee all students, employees, and applications, with the ability to update statuses, assign tasks, and manage user access.
- Real-time Chatbot: A dedicated chatbot facilitates communication between employees and admins regarding specific student applications.
- Customizable Forms: Admins can add and customize application questions to tailor the submission process for new students or university requirements. A tool will decide how to incorporate newly gathered information into an output.
- WhatsApp Notifications: Admins receive immediate notifications via WhatsApp for new application requests, and employees receive updates from admins through WhatsApp as well.
- Student Record Sharing Detection: The system identifies potential duplicate student records across different employees using email addresses and phone numbers.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust, stability, and competence in handling important application processes.
- Background color: Very light blue (#E8EAF6) to create a calm and professional environment.
- Accent color: Purple (#7E57C2) to highlight calls to action and important updates.
- Body and headline font: 'Inter', a sans-serif font that offers a clean and modern look, suitable for both headlines and body text.
- Use clear and intuitive icons to represent application statuses, document types, and communication channels.
- Design a clean and organized layout that prioritizes important information, with easy navigation for both admin and employee users.
- Incorporate subtle transitions and feedback animations to enhance user experience and provide clear visual cues for actions and updates.
